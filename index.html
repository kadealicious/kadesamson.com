Westy - 2021-2023

C/OpenGL Rendering Engine

This project is a forward+ rendering engine that I wrote entirely in C using OpenGl. Westy supports .obj model loading, normal mapping, deferred AND forward rendering for the best performance possible. The engine was originally written using the Windows-default HWND windowing system and CGLM for linear algebra operations, with graphics being done via pixel blitting on the CPU. I am currently rewriting the engine yet again to use Vulkan for improved performance and more control over the entire rendering pipeline.



JS Popup - 2021

Responsive, Unintrusive Popups in JavaScript

This script is used to create an unintrusive popup element in an HTML document. The design goal here was to create a popup element that, A: Didn’t bloat the section of code it was applied to within the HTML document and, B: Was easily customizable and felt natural to use. I am very satisfied with how this project came out!
